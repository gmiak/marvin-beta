# marvin-beta
Marvin-beta är en chattbot som kan svara på "alla" dina frågor.

Man startar programmet genom att köra kommando :
* python3 main.py
Sen kommer en main-meny för att kunna kommunicera med Marvin-beta.

Det finns en till funktion som Marvin-beta har och man kommer åt det genom att skicka en parameter till programmet,
testa kommando "-h" för att få en skarp beskrivning om funtionen. Gör så här :

* python3 main.py -h
